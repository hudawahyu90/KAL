Pengantar Transformasi Linier#

Fungsi T yang memetakan ruang vektor V ruang vektor W:




V: domain T
W: codomain T
Suatu fungsi dari R2 ke R2

Solusi :

Linier Transformation
V, W ruang vektor
T : V -> maka V ke W transformasi linier
Notes :
(1) A linear transformation dikatan operation yang mempertahankan aljabar linier (lihat catatan).
(2) Transformasi linier dari T : V -> V disebut operator linear
Latihan 2: (Verifikasi transformasi linier T dari R2 ke R2)
T(v1, v 2) = (v 1 - v2,v1 + 2v2)

Bukti

u = (u1, u2), v = (v1, v2) : vektor dalam R2 , c : bilangan real
(1) penjumlahan Vektor:
u + v = (u1, u2) + (v1, v2) = (u1+v1,u2+v2)


(2) Perkalian skalar
cu = c(u1, u2) = (cu1, cu2)

sehingga, T adalah Transformasi linier
3: (fungsi fungsi yang bukan transformasi linear)

Tugas #

Soal
Di taruh di web statis membuktikan bahwa T(v1,v2) = (v1 + v2, v1)
buat contoh 2 titik masing masing
Jawab
1. Membuktikan bahwa transformasi T(v1,v2) = (v1 + v2, v1) adalah transformasi linier dengan menggunakan dua vektor.
misalnya:
- u = (1,2)
- v = (3,4)

2. Uji Penjumlahan untuk mengecek apakah
T(u + v) = T(u) + T(v)
- u + v = ((1+3),(2+4)) = (4,6)
- T(4,6) = ((4+6), 4) = (10,4)
lalu hitung masing-masing:
- T(1,2) = ((1+2),1) = (3,2)
- T(3,4) = ((3+4),3) = (7,3)
- T(1,2) + T(3,4) = ((3+7),(1+3)) = (10,4)
Sama, jadi sifat penjumlahannya terpenuhi

3. Uji Perkalian Skalar untuk mengecek apakah
T(cu) = cT(u)
- cu = 2 * (1,2) = (2,4)
- T(2,4) = ((2+4),2) = (6,2)
lalu: ,
- T(1,2) = (3,1)
- 2T(1,2) = 2*(3,1) = (6,2)
sama, jadi sifat perkalian skalar juga terpenuhi.
karena kedua sifat transformasi linier (penjumlahan dan perkalian skalar) terpenuhi, maka
T(v1,v2) = (v1 + v2, v1)

Untuk geogebranya

1. Refleksi about the x- axis dengan matriks {{1,0},{0,-1}}
2. Reflection about the y- axis dengan matriks {{-1,0},{0,1}}
3. Reflection about line x = y dengan matriks {{0,1},{1,0}}
4. Reflection about the line y = -x {{0,-1},{-1,0}}
5. Reflection about the origin {{-1,0},{0,-1}}
