Aljabar Matriks#

Tujuan Pembelajaran#

-Menyatakan data dalam bentuk matriks
-Memahami konsep matriks dan jenis-jenisnya
-Melakukan operasi pada matriks, seperti perkalian, penjumlahan pengurangan dan transposisi
-Menyelesaikan masalah yang berkaitan dengan perkalian matriks
-Menggunakan matriks untuk menyelesaikan masalah matematika, seperti persamaan linear dan transformasi linear
-Menggunakan determinan matriks untuk memecahkan masalah kehidupan sehari-hari
-Menuliskan permasalahan nyata dalam bentuk matriks
-Menjelaskan sifat-sifat perkalian matriks
-Menjelaskan kesamaan matriks.

Tiga Jenis Operasi Matriks#

Operasi Penjumlahan Matriks
Misalkan terdapat dua buah matriks, yaitu matriks A dan matriks B. Jika matriks C adalah matriks penjumlahan dari A dengan B, maka matriks C dapat diperoleh dengan menjumlahkan setiap elemen pada matriks A yang seletak dengan setiap elemen pada matriks B. Oleh karena itu, syarat agar dua atau lebih matriks dapat dijumlahkan adalah harus memiliki ordo yang sama.
sifat penjumlahan :
-Komunikatif = 𝐴+𝐵=𝐵+𝐴
-Asosiatif:𝐴+(𝐵+𝐶)=(𝐴+𝐵)+𝐶

Operasi Pengurangan Matriks
Misalkan terdapat dua buah matriks, yaitu matriks A dan matriks B. Jika matriks C adalah matriks pengurangan dari A dengan B, maka matriks C dapat diperoleh dengan mengurangkan setiap elemen pada matriks A yang seletak dengan setiap elemen pada matriks B.
Sifat perkalian =
-Asosiatif: 𝐴(𝐵𝐶)=(𝐴𝐵)𝐶
-Distributif Kiri: 𝐴(𝐵+𝐶)=𝐴𝐵+𝐴𝐶
-Distributif Kanan: (𝐵+𝐶)𝐴=𝐵𝐴+𝐶𝐴

Operasi Perkalian Matriks
Operasi perkalian matriks dibagi menjadi dua nih, yaitu perkalian matriks dengan bilangan real (skalar) dan perkalian antarmatriks.
Sifat perkalian skalar =
Distributif Matriks: 𝑎(𝐵+𝐶)=𝑎𝐵+𝑎𝐶
Distributif Skalar: (𝑎+𝑏)𝐶=𝑎𝐶+𝑏𝐶
Asosiatif Skalar: 𝑎(𝑏𝐶)=(𝑎𝑏)𝐶
Komutatif Skalar: 𝑎(𝐵𝐶)=(𝑎𝐵)𝐶=𝐵(𝑎𝐶)

Tugas #

Soal

Menghitung persamaan 3 variable dengan hasil invers 1,0,0

Jawab
Langkah 1: Buat Persamaan 3x3

Misalkan kita memiliki sistem persamaan linear 3x3 berikut:

< 2 x + 3 y + z = 8 4 x + y + 2 z = 9 3 x + 2 y + 3 z = 7

Matriks koefisien dari sistem ini adalah:

A = [ 2 3 1 4 1 2 3 2 3 ]

Dan matriks hasilnya adalah:

B = [ 8 9 7 ]
Langkah 2: Bentuk Augmented Matrix

Gabungkan matriks A dengan matriks identitas I untuk membentuk augmented matrix:

[ A ∣ I ] = [ 2 3 1 1 0 0 4 1 2 0 1 0 3 2 3 0 0 1 ]
Langkah 3: Buat Elemen (1,1) Menjadi 1

Saat ini, elemen pertama (1,1) adalah 2. Kita ubah menjadi 1 dengan membagi baris pertama dengan 2:

R 1 ← 1 2 R 1 [ 1 1,5 0,5 0,5 0 0 4 1 2 0 1 0 3 2 3 0 0 1 ]
Langkah 4: Nol-kan Elemen di Bawah (1,1)

Langkah 4: Nol-kan Elemen di Bawah (1,1) Sekarang kita nol-kan elemen di bawah elemen (1,1) (yaitu elemen di baris kedua dan ketiga, kolom pertama)

[ 1 1,5 0,5 0,5 0 0 0 -5 0 -2 1 0 0 -2,5 1,5 -1,5 0 1 ]
Langkah 5: Buat Elemen (2,2) Menjadi 1

Sekarang kita buat elemen (2,2) menjadi 1 dengan membagi baris kedua dengan -5:

[ 1 1,5 0,5 0,5 0 0 0 1 0 0,4 -0,2 0 0 -2,5 1,5 -1,5 0 1 ]
Langkah 6: Nol-kan Elemen di Bawah (2,2)

Sekarang kita nol-kan elemen (3,2) (yaitu elemen di baris ketiga, kolom kedua)Maka augmented matrix menjadi:

[ 1 1,5 0,5 0,5 0 0 0 1 0 0,4 -0,2 0 0 0 1,5 -0,5 0,5 1 ]
Langkah 7: Buat Elemen (3,3) Menjadi 1

Sekarang kita buat elemen (3,3) menjadi 1 dengan membagi baris ketiga dengan 1.5:

[ 1 1,5 0,5 0,5 0 0 0 1 0 0,4 -0,2 0 0 0 1 − 1 3 1 3 2 3 ]
Langkah 8: Nol-kan Elemen di Atas (3,3)
[ 1 1,5 0,5 -0,6667 -0,3333 -0,3333 0 1 0 0,4 -0,2 0 0 0 1 − 1 3 1 3 2 3 ]
Langkah 9: Hitung A − 1 × B
A − 1 × B = [ 0,0667 − 0,3333 -0,3333 0,4 -0,2 0 − 1 3 1 3 2 3 ] × [ 8 9 7 ]

Hasilnya Adalah:

A − 1 × B = [ 8 9 7 ]
